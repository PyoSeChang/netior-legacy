How To Read This

  • All concepts keep the same high-level constraint: minimal taeguk mood, helix or relational twist, and a single central Netior dot.
  • This page is intentionally broad. The goal is to generate shape vocabulary first, then narrow down to a smaller brand family.
  • The newest concepts bias toward crossing energy, because that feels closer to co-evolution and relational rewiring than clean parallelism.
  • Use the palette selector on each card to test whether the symbol reads as a tool, interface, instrument, ritual mark, or generic tech badge.
  • Color is doing a lot of symbolic work. If a form feels right but the meaning feels wrong, change palette first before discarding the concept.
